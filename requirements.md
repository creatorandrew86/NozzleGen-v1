# Software Requirements
In order to run NozzleGen you need to have installed the following:
  - Python 3.10 or newer
  - The following libraries also need to be installed
    - Matplotlib
    - Cadquery (optional)
    - Tkinter (likely already part of the Python standard library)
    - Pandas
    - Numpy
    - Scipy
    - Pillow (PIL)
      
To install the libraries you can run:
```
pip install matplotlib cadquery pandas numpy scipy pillow
```
