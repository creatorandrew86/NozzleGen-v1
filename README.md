# NozzleGen-v1
Bell and Conical nozzle contour generation algorithm and User Interface

# How To Run The Program
Step 1: Check the requirements.md file and make sure you have all of them installed
Step 2: Download the Nozzle Gen_v1 folder. Make sure all the files retain the name from the folder (if you do not want to change the code)
Step 3: Run the "compiler.py" file
